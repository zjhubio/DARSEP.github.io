Local Installation & Usage

In order to maximize the user experience, we have prepared a convenient local installation package with a detailed installation and operation guide to ensure that users can easily complete the installation and get started for use immediately. The following content will guide you through the entire process.

Pre-installation Preparation
  • First of all, please prepare a pristine python environment.
  • Before installing this package, please follow the requirements.txt file in Github to install the necessary project dependencies.
Installation
Use the following command to install the DARSEP tool:
pip install DARSEP -i https://pypi.python.org/simple

Usage

After installing the tool, you can call the method directly in the code, there are two interfaces, DARSEP-SPRLM and DARSEP-PMLM, the following describes the usage of these two interfaces:

The first thing you can do is to import the DARSEP package directly:
import DARSEP

  • DARSEP-SPRLM
    • The DARSEP-SPRLM method aims to optimize the sequences using reinforcement learning, after running it will get the optimized sequences as well as the values of the predicted fitness, which will be saved to a csv file, which can be implemented by running the following code:
    from DARSEP import DARSEP_SPRLM
    DARSEP_SPRLM.selfPlay()

  • DARSEP-PMLM
    • DARSEP-PMLM is designed to provide an interface to downstream task analysis methods for SRAS-CoV-2 spike protein sequences, and is used as described below:
    1. Clustering analysis: Users can cluster the customised SRAS-CoV-2 sequences by calling the method, though it may take some time, using the following method:
    from DARSEP import DARSEP_PMLM
    DARSEP_SPRLM.Cluster(fasta_file)
    where fasta_file is the path to the fasta file.

    2. Evolutionary field construction: The user can construct the vector field of the virus sequence by calling the method as follows:
    from DARSEP import DARSEP_PMLM
    DARSEP_SPRLM.evo(fasta_file)
    where fasta_file is the path to the fasta file.

    3. Missense Mutation Analysis: Users can call the method to perform missense mutation analysis on wild-type sequences as follows:
    from DARSEP import DARSEP_PMLM
    DARSEP_SPRLM.MissenseAnalysis()

    4. Grammatical semantic analysis: Users can analyze the changes in the semantic syntax of a particular SRAS-CoV-2 according to its mutation site invocation method as follows:
    from DARSEP import DARSEP_PMLM
    DARSEP_SPRLM.vocs(mutations)
    Suggested input format for mutations: ['G339D','S371L','S373P','S375F','K417N','N440K','G446S','S477N','T478K','E484A','Q493R','G496S','Q498R','N501Y','Y505H']
